# SOC Lab

Cyber Defense Lab / Mini SOC environment built for learning and portfolio purposes.

This project simulates a small enterprise infrastructure monitored by a SOC (Security Operations Center).  
The goal is to practice **detection engineering, attack simulation, log analysis, and automation**.

---

# Objectives

This lab aims to demonstrate skills in:

- Security monitoring
- SIEM deployment
- Network detection
- Attack simulation
- Incident investigation
- Detection engineering
- Infrastructure automation

The environment is designed to simulate **real SOC workflows**.

---

# Architecture

The lab is based on a segmented network architecture.

Network zones include:

- Attacker network (Kali Linux)
- DMZ (public services)
- Internal network
- SOC monitoring network
